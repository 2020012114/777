## 一.信息素养

​        信息素养既是一种基本能力，又是一种对信息社会的适应能力。包括：

1. **基本学习技能（指读、写、算）**
2. **信息素养**
3. **创新思维能力**
4. **人际交往与合作能力**

## 三.研究层面

### 1.从当代大学生来讲

我们日常所接触的信息很丰富，能够看到各种各样的东西。我们在浏览网页的时候常常走马观花，并不能很好的学习利用信息，而素养实践正是给我们提供了这个途径，让我们在真正意义上的获取了知识。

### 2.从社会来讲

在繁杂的社会里，一些虚假的信息成为了人们不安的源泉，而信息素养实践正是给了我们不信谣传谣的机会。有利于社会更和谐的发展。
